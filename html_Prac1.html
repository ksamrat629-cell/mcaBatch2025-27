Features To Recreate

  • Media elements: <figure>, <img>, <figcaption>

Steps (Ordered)

  1. Create the overall structure and headings.
  2. Add navigation links that jump to each section.
  3. Place images and captions using <figure>.
  4. Build a table with a header row.
  5. Make a contact form with labels and inputs.
  6. Enhance with <details>, <progress>, and <meter>.
